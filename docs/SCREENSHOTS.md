# Screenshots

This page collects screenshots used in the README and guides. Capture new ones at a 1280×820 window for consistency, and save them in `assets/`.

> **Note:** the in-app docs viewer renders relative image links from `docs/`, so paths in markdown should be like `../assets/home.png`.

## How to update

1. Capture the screenshot on a 1280×820 window.
2. Save as PNG into `assets/` with a descriptive lowercase name (e.g. `home_dashboard.png`).
3. Reference it from this page **and** wherever else it's used.
4. Replace any stale captures — don't accumulate duplicates.
